# ecommerce-analytics-sql
An end-to-end SQL analytics project using PostgreSQL to perform exploratory data analysis, advanced analytics, customer and product reporting, and generate actionable business insights from e-commerce sales data.

## Overview
This project demonstrates SQL analytics expertise using an existing data warehouse. 
Focus: Exploratory Data Analysis, Advanced Analytics, and Business Reporting Views.

## What I Did (3 Components)

### 1. Exploratory Data Analysis (EDA)
- Database exploration (table, data quality)
- Dimension exploration (countries, categories)
- Date exploration
- Measures exploration (sales, quantities)
- Ranking Analysis (Top/bottom performer ranking)

### 2. Advanced Analytics
- Sales trends over time
- Cumulative analysis
- Performance analysis
- Part-to-Whole Analysis (Proportional)
- Data segmentation

### 3. Production Views
- `gold.report_customers` — Customer metrics + segmentation
- `gold.report_products` — Product metrics + segmentation

## Data Source
Used existing PostgreSQL data warehouse with:
- Fact table: Orders (18,484 records)
- Dimension tables: Customers, Products

## Key Skills Demonstrated
- Complex SQL queries (CTEs, window functions, aggregations)
- Business intelligence thinking
- Data exploration methodology
- Professional documentation
